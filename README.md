## CI/CD Pipeline

This project demonstrates an automated CI/CD workflow using GitHub Actions and Amazon ECR.

### Pipeline Flow

Developer
↓
Git Push
↓
GitHub Actions
↓
Docker Image Build
↓
Amazon ECR Authentication
↓
Docker Image Push
↓
Amazon ECR

### Skills Demonstrated

* GitHub Actions
* Docker
* Amazon ECR
* AWS IAM
* Secrets Management
* Continuous Integration
* Continuous Delivery
* Platform Engineering
